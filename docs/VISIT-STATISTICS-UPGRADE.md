## 🌍 部署指南

### 环境要求
- Node.js 18+
- PostgreSQL 数据库 (Neon)
- Vercel 账号

### 部署步骤
1. **克隆项目**
   ```bash
   git clone <repository>
   cd love-website
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置环境变量**
   ```bash
   cp .env.deployment.template .env.local
   # 填入数据库连接信息
   ```

4. **构建项目**
   ```bash
   npm run build
   ```

5. **部署到 Vercel**
   ```bash
   vercel --prod
   ```

### 环境变量配置
```env
DATABASE_URL=postgresql://...
REDIS_HOST=...
REDIS_PORT=...
REDIS_PASSWORD=...
NODE_ENV=production
```
